Por que Estudar JavaScript?

JavaScript é uma das 3 linguagens que todos os desenvolvedores web devem aprender:

1. HTML

Para definir o conteúdo das páginas web

2. CSS

Para especificar o layout das páginas web

3. JavaScript

Para programar o comportamento das páginas web

Você Sabia?

  • ✓ JavaScript e Java são linguagens completamente diferentes, tanto em conceito quanto em design.
  • ✓ JavaScript foi inventado por Brendan Eich em 1995 e se tornou um padrão ECMA em 1997.
  • ✓ ECMA-262 é o nome oficial do padrão. ECMAScript é o nome oficial da linguagem.
