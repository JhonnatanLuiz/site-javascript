JavaScript Map Methods

Nesta página, você aprende os métodos mais usados do Map: set, get, has, delete, clear, e os iteradores keys/values/entries.

Dica: map.set() retorna o próprio Map, então dá para encadear chamadas.

Tabela rápida

Método Uso Retorno
set(k, v) Define/atualiza valor Map (encadeável)
get(k) Lê valor pela chave Valor ou undefined
has(k) Verifica existência boolean
delete(k) Remove chave boolean
clear() Limpa tudo undefined
keys() Itera chaves Iterator
values() Itera valores Iterator
entries() Itera pares [k, v] Iterator
forEach(fn) Itera via callback undefined
