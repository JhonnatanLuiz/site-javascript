JavaScript Statements

Exemplo - Programa JavaScript

let x, y, z;    // Statement 1
x = 5;          // Statement 2
y = 6;          // Statement 3
z = x + y;      // Statement 4

Programas JavaScript

Um programa de computador é uma lista de "instruções" a serem "executadas" por um computador.

Essas instruções de programação são chamadas de statements (declarações/instruções).

A maioria dos programas JavaScript contém muitos statements.

📝 Como Funcionam os Statements:

  • 1. Os statements são executados, um por um
  • 2. Na mesma ordem em que são escritos
  • 3. Em HTML, programas JavaScript são executados pelo navegador

Composição de Statements JavaScript

JavaScript statements são compostos de:

📌

Valores

Literais e variáveis

➕

Operadores

+, -, *, /, =, etc.

🔢

Expressões

Combinações que computam valores

🔑

Palavras-chave

let, const, if, for, etc.

💬

Comentários

// ou /* */

Exemplo - Statement Completo

Este statement diz ao navegador para escrever "Hello Dolly." dentro de um elemento HTML com id="demo":

document.getElementById("demo").innerHTML = "Hello Dolly.";

💡 Nota: Programas JavaScript (e JavaScript statements) são frequentemente chamados de código JavaScript.

Ponto e Vírgula ;

Ponto e vírgula ; separa statements JavaScript.

Adicione um ponto e vírgula ao final de cada statement executável:

Exemplo - Statements em Linhas Separadas

let a, b, c;  // Declare 3 variáveis
a = 5;        // Atribui o valor 5 para a
b = 6;        // Atribui o valor 6 para b
c = a + b;    // Atribui a soma de a e b para c

Exemplo - Múltiplos Statements na Mesma Linha

Quando separados por ponto e vírgula, múltiplos statements em uma linha são permitidos:

a = 5; b = 6; c = a + b;

⚠️ Ponto e Vírgula é Opcional, mas ALTAMENTE RECOMENDADO

Na web, você pode ver exemplos sem ponto e vírgula. Finalizar statements com ponto e vírgula não é obrigatório, mas é uma boa prática recomendada.

✓ Boas práticas:

  • Sempre use ponto e vírgula ao final dos statements
  • Evita comportamentos inesperados
  • Torna o código mais legível e profissional

Espaços em Branco JavaScript

JavaScript ignora múltiplos espaços. Você pode adicionar espaços em branco ao seu script para torná-lo mais legível.

Exemplo - Espaços em Branco

As linhas seguintes são equivalentes:

let person = "Hege";
let person="Hege";

Ambas resultam em: person = "Hege"

💡 Boa Prática:

Coloque espaços ao redor dos operadores ( = + - * / ):

let x = y + z;

Comprimento de Linha e Quebras de Linha

Para melhor legibilidade, programadores geralmente gostam de evitar linhas de código maiores que 80 caracteres.

Se um statement JavaScript não cabe em uma linha, o melhor lugar para quebrá-lo é depois de um operador:

Exemplo - Quebra de Linha

document.getElementById("demo").innerHTML =
"Hello Dolly!";

📏 Regra de Legibilidade:

  • ✓ Mantenha linhas com menos de 80 caracteres
  • ✓ Quebre linhas depois de operadores
  • ✓ Use indentação consistente

Blocos de Código JavaScript

JavaScript statements podem ser agrupados em blocos de código, dentro de chaves {...}.

O propósito dos blocos de código é definir statements para serem executados juntos.

Um lugar onde você encontrará statements agrupados em blocos é nas funções JavaScript:

Exemplo - Bloco de Código em Função

function myFunction() {
  document.getElementById("demo1").innerHTML = "Hello Dolly!";
  document.getElementById("demo2").innerHTML = "How are you?";
}

📦 Blocos de Código:

  • • Usado em funções, loops e condicionais
  • • Delimitado por chaves { }
  • • Neste tutorial usamos 2 espaços de indentação
  • • Você aprenderá mais sobre funções depois neste tutorial

Palavras-Chave JavaScript

JavaScript statements frequentemente começam com uma palavra-chave para identificar a ação JavaScript a ser executada.

Aqui está uma lista de algumas palavras-chave que você aprenderá neste tutorial:

Palavra-Chave Descrição
var Declara uma variável
let Declara uma variável de bloco
const Declara uma constante de bloco
if Marca um bloco de statements para ser executado sob uma condição
switch Marca um bloco de statements para ser executado em diferentes casos
for Marca um bloco de statements para ser executado em um loop
function Declara uma função
return Sai de uma função
try Implementa tratamento de erros em um bloco de statements

⚠️ Palavras Reservadas:

Palavras-chave JavaScript são palavras reservadas. Palavras reservadas não podem ser usadas como nomes de variáveis.

❌ let let = 5; // ERRO!

✓ let myVariable = 5; // Correto
