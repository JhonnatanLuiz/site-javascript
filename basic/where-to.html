JavaScript Where To

A Tag <script>

Em HTML, o código JavaScript é inserido entre as tags <script> e </script>.

Exemplo

<script>
document.getElementById("demo").innerHTML = "Meu Primeiro JavaScript";
</script>

Clique no botão acima!

📝 Observação: Exemplos antigos de JavaScript podem usar um atributo type: <script type="text/javascript">. O atributo type não é necessário. JavaScript é a linguagem de script padrão em HTML.

Funções e Eventos JavaScript

Uma function JavaScript é um bloco de código JavaScript que pode ser executado quando "chamado".

Por exemplo, uma função pode ser chamada quando um evento ocorre, como quando o usuário clica em um botão.

💡 Você aprenderá muito mais sobre funções e eventos em capítulos posteriores.

JavaScript em <head> ou <body>

Você pode colocar qualquer número de scripts em um documento HTML.

Os scripts podem ser colocados no <body>, ou na seção <head> de uma página HTML, ou em ambos.

JavaScript em <head>

Neste exemplo, uma function JavaScript é colocada na seção <head> de uma página HTML.

A função é invocada (chamada) quando um botão é clicado:

Exemplo

<!DOCTYPE html>
<html>
<head>
<script>
function myFunction() {
  document.getElementById("demo").innerHTML = "Parágrafo alterado.";
}
</script>
</head>
<body>

<h2>JavaScript no Head</h2>
<p id="demo">Um Parágrafo</p>
<button onclick="myFunction()">Clique aqui</button>

</body>
</html>

Um Parágrafo

JavaScript em <body>

Neste exemplo, uma function JavaScript é colocada na seção <body> de uma página HTML.

A função é invocada (chamada) quando um botão é clicado:

Exemplo

<!DOCTYPE html>
<html>
<body>

<h2>JavaScript no Body</h2>
<p id="demo">Um Parágrafo</p>
<button onclick="myFunction()">Clique aqui</button>

<script>
function myFunction() {
  document.getElementById("demo").innerHTML = "Parágrafo alterado.";
}
</script>

</body>
</html>

Um Parágrafo

⚡ Dica de Performance: Colocar scripts na parte inferior do elemento <body> melhora a velocidade de exibição, porque a interpretação de scripts diminui a velocidade de exibição.

JavaScript Externo

Os scripts também podem ser colocados em arquivos externos:

Arquivo Externo: myScript.js

function myFunction() {
  document.getElementById("demo").innerHTML = "Parágrafo alterado.";
}

Scripts externos são práticos quando o mesmo código é usado em muitas páginas web diferentes.

Arquivos JavaScript têm a extensão .js

Para usar um script externo, coloque o nome do arquivo de script no atributo src (source) de uma tag <script>:

Exemplo

<script src="myScript.js"></script>

✅ Você pode colocar uma referência de script externo em <head> ou <body> como desejar.

🚫 Scripts externos não podem conter tags <script>.

Vantagens do JavaScript Externo

Colocar scripts em arquivos externos tem algumas vantagens:

📁

Separação de Código

Separa HTML e código JavaScript

📖

Facilita Leitura

Torna HTML e JavaScript mais fáceis de ler e manter

⚡

Cache e Performance

Arquivos JavaScript em cache podem acelerar o carregamento de páginas

💡 Múltiplos Scripts: Para adicionar vários arquivos de script em uma página - use várias tags script:

<script src="myScript1.js"></script>
<script src="myScript2.js"></script>

Referências Externas

Um script externo pode ser referenciado de 3 maneiras diferentes:

1️⃣ Com uma URL completa (endereço web completo)

<script src="https://www.w3schools.com/js/myScript.js"></script>

2️⃣ Com um caminho de arquivo (como /js/)

<script src="/js/myScript.js"></script>

3️⃣ Sem nenhum caminho

<script src="myScript.js"></script>
