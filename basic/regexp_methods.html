RegExp Methods

Os métodos mais comuns para trabalhar com regex em JavaScript aparecem tanto em RegExp quanto em String, como: test, exec, match, replace.

W3Schools: JavaScript RegExp Methods

Checklist mental

  • Validar? use regex.test(texto).
  • Extrair? use texto.match(regex) ou regex.exec(texto).
  • Substituir? use texto.replace(regex, novo).
