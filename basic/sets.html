JavaScript Sets

Um Set em JavaScript é uma coleção de valores únicos. Cada valor pode ocorrer apenas uma vez em um Set.

Os valores de um Set podem ser de qualquer tipo: primitivos ou objetos.

Criando um Set

Exemplo 1: Criar um Set vazio

// Criar um Set vazio
const mySet = new Set();

// Adicionar valores
mySet.add("a");
mySet.add("b");
mySet.add("c");

Exemplo 2: Criar Set a partir de um array

// Passar um array para new Set()
const letters = new Set(["a", "b", "c"]);

console.log(letters.size); // 3

Método add()

Exemplo 3: Adicionar valores ao Set

const mySet = new Set();

mySet.add("apple");
mySet.add("banana");
mySet.add("orange");
mySet.add("apple"); // Duplicado - não será adicionado

console.log(mySet.size); // 3 (não 4)

Método has()

O método has() retorna true se um valor especificado existe no Set.

Exemplo 4: Verificar se valor existe

const fruits = new Set(["apple", "banana", "orange"]);

console.log(fruits.has("apple"));  // true
console.log(fruits.has("grape"));  // false

Iterando Sobre um Set

Você pode iterar sobre os elementos de um Set usando forEach():

Exemplo 5: forEach() em um Set

const letters = new Set(["a", "b", "c"]);

letters.forEach(function(value) {
  console.log(value);
});

Exemplo 6: Loop for...of

const numbers = new Set([1, 2, 3, 4, 5]);

for (const number of numbers) {
  console.log(number);
}

Exemplo 7: Método values()

const mySet = new Set(["x", "y", "z"]);

// Obter um iterator dos valores
const iterator = mySet.values();

for (const value of iterator) {
  console.log(value);
}

Casos de Uso

Um dos usos mais comuns de Sets é remover duplicatas de arrays:

Exemplo 8: Remover duplicatas

const numbers = [1, 2, 2, 3, 4, 4, 5];

// Converter para Set e de volta para Array
const unique = [...new Set(numbers)];

console.log(unique); // [1, 2, 3, 4, 5]

Exemplo 9: Sets com diferentes tipos

const mySet = new Set();

// Adicionar diferentes tipos
mySet.add(1);
mySet.add("Hello");
mySet.add(true);
mySet.add({name: "John"});
mySet.add([1, 2, 3]);

Set vs Array

Característica Set Array
Valores únicos ✓ Sim ✗ Não
Ordem preservada ✓ Sim (ordem de inserção) ✓ Sim
Acesso por índice ✗ Não ✓ Sim
Performance (busca) O(1) - Rápido O(n) - Lento
Tamanho .size .length
Métodos disponíveis add, delete, has, clear push, pop, splice, map, filter...

🔑 Pontos importantes sobre Sets:

  • ✓ Valores únicos: Sets automaticamente removem duplicatas
  • ✓ Qualquer tipo: Pode armazenar qualquer tipo de valor JavaScript
  • ✓ Ordem de inserção: Sets mantêm a ordem em que os valores foram adicionados
  • ✓ Performance: Verificação de existência (has) é muito rápida - O(1)
  • ✓ Iterável: Pode usar forEach, for...of e spread operator (...)

Métodos Essenciais

Método Descrição
add(value) Adiciona um novo elemento ao Set
delete(value) Remove um elemento do Set
has(value) Verifica se um valor existe no Set
clear() Remove todos os elementos do Set
forEach() Invoca uma função para cada elemento
values() Retorna um Iterator com os valores
keys() Igual a values() (para compatibilidade com Map)
entries() Retorna um Iterator com pares [value, value]
