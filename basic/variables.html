JavaScript Variables

Variáveis = Contêineres de Dados

Variáveis JavaScript são contêineres para armazenar dados.

Variáveis JavaScript podem ser declaradas de 4 maneiras:

✅ JavaScript Moderno

  • • Usando let
  • • Usando const

⚠️ JavaScript Antigo

  • • Usando var (Não Recomendado)
  • • Automaticamente (Não Recomendado)

Exemplo usando let

let x = 5;
let y = 6;
let z = x + y;

Exemplo usando const

const x = 5;
const y = 6;
const z = x + y;

📦 Dos exemplos você pode deduzir:

  • • x contém (ou armazena) o valor 5
  • • y contém (ou armazena) o valor 6
  • • z contém (ou armazena) o valor 11

Identificadores JavaScript

Variáveis são identificadas com nomes únicos chamados identificadores.

Nomes podem ser curtos como x, y, z.

Nomes podem ser descritivos como age, sum, carName.

📋 Regras para construir nomes (identificadores):

  • ✓ Nomes podem conter letras, dígitos, underscores (_) e cifrões ($)
  • ✓ Nomes devem começar com uma letra, um $ ou um underscore (_)
  • ⚠ Nomes são case sensitive (X é diferente de x)
  • ✗ Palavras reservadas (palavras-chave JavaScript) não podem ser usadas como nomes

⚠️ Nota: Números não são permitidos como primeiro caractere em nomes.

Dessa forma, JavaScript pode facilmente distinguir identificadores de números.

JavaScript Underscore (_)

JavaScript trata underscore como uma letra.

Identificadores contendo _ são nomes de variáveis válidos:

Exemplo - Underscore

let _lastName = "Johnson";
let _x = 2;
let _100 = 5;

💡 Convenção: Uma convenção entre programadores profissionais é começar um nome com underscore para variáveis "privadas".

JavaScript Dollar Sign $

JavaScript também trata o cifrão como uma letra.

Identificadores contendo $ são nomes de variáveis válidos:

Exemplo - Dollar Sign

let $ = "Hello World";
let $$$ = 2;
let $myMoney = 5;

💡 Uso Profissional: Usar $ não é muito comum em JavaScript, mas programadores profissionais frequentemente o usam como alias para a função principal em bibliotecas JavaScript (como jQuery).

Declarando Variáveis JavaScript

Criar uma variável em JavaScript é chamado de declarar uma variável.

Você declara uma variável JavaScript com a palavra-chave let ou const.

Declarando uma Variável usando let

let carName;

Após a declaração, a variável não tem valor (tecnicamente é undefined).

Para atribuir um valor à variável, use o sinal de igual:

carName = "Volvo";

Na maioria das vezes, você atribuirá um valor à variável quando declará-la:

let carName = "Volvo";

Declarando uma Variável usando const

Sempre use const se o valor não deve ser alterado:

const carName = "Volvo";

Exemplo Misto:

const price1 = 5;
const price2 = 6;
let total = price1 + price2;

As variáveis price1 e price2 são declaradas com const. Seus valores não podem ser alterados. A variável total é declarada com let e pode ser alterada.

Declarando Variável Automaticamente

Variáveis não declaradas são automaticamente declaradas quando usadas pela primeira vez:

Exemplo (Não Recomendado)

x = 5;
y = 6;
z = x + y;

⚠️ NÃO RECOMENDADO!

É uma boa prática de programação declarar todas as variáveis no início de um script.

Declarando Variável usando var

A palavra-chave var era usada em todo código JavaScript antes de 2015.

As palavras-chave let e const foram adicionadas ao JavaScript em 2015.

Usando var (Não Recomendado)

var x = 5;
var y = 6;
var z = x + y;

✅ Quando Usar var, let ou const?

  1. Sempre declare variáveis
  2. Sempre use const se o valor não deve ser alterado
  3. Sempre use const se o tipo não deve ser alterado (Arrays e Objects)
  4. Use let apenas se você não puder usar const
  5. Nunca use var se puder usar let ou const

Tipos de Dados JavaScript

Variáveis JavaScript podem conter 8 tipos de dados, mas por enquanto, pense apenas em números e strings.

Strings são texto escrito entre aspas.

Números são escritos sem aspas.

Se você colocar um número entre aspas, ele será tratado como uma string de texto.

Exemplo - Tipos de Dados

const pi = 3.14;
let person = "John Doe";
let answer = 'Yes I am!';

Uma Declaração, Muitas Variáveis

Você pode declarar muitas variáveis em uma declaração.

Comece a declaração com let ou const e separe as variáveis por vírgula:

Em Uma Linha

let person = "John Doe", carName = "Volvo", price = 200;

Múltiplas Linhas

let person = "John Doe",
    carName = "Volvo",
    price = 200;

O Operador de Atribuição

Em JavaScript, o sinal de igual (=) é um operador de atribuição, não um operador "igual a".

Isso é diferente da álgebra. O seguinte não faz sentido na álgebra:

x = x + 5

Em JavaScript, no entanto, faz todo sentido: ele atribui o valor de x + 5 para x.

(Calcula o valor de x + 5 e coloca o resultado em x. O valor de x é incrementado em 5.)

💡 Nota: O operador "igual a" é escrito como == em JavaScript.

Aritmética JavaScript

Como na álgebra, você pode fazer aritmética com variáveis JavaScript, usando operadores como = e +:

Somando Números

let x = 5 + 2 + 3;

Concatenando Strings

let x = "John" + " " + "Doe";

⚠️ Nota Importante:

Se você colocar um número entre aspas, o resto dos números será tratado como strings e concatenado.

let x = "5" + 2 + 3;

Resultado: "523"

let x = 2 + 3 + "5";

Resultado: "55"
