JavaScript Set Methods

JavaScript Sets possuem vários métodos úteis para adicionar, remover e verificar elementos.

Método add()

O método add() adiciona um novo elemento ao Set. Retorna o próprio Set, permitindo encadeamento.

Exemplo 1: Encadeando add()

const mySet = new Set();

// Encadear chamadas add()
mySet.add(1).add(2).add(3).add(4).add(5);

console.log(mySet); // Set {1, 2, 3, 4, 5}

Método delete()

O método delete() remove um elemento específico do Set. Retorna true se o elemento foi removido, false se não existia.

Exemplo 2: Removendo elementos

const fruits = new Set(["apple", "banana", "orange"]);

fruits.delete("banana");

console.log(fruits.has("banana")); // false
console.log(fruits.size);              // 2

Método clear()

O método clear() remove todos os elementos do Set.

Exemplo 3: Limpando um Set

const numbers = new Set([1, 2, 3, 4, 5]);

console.log(numbers.size); // 5

numbers.clear();

console.log(numbers.size); // 0

Método has()

O método has() verifica se um valor existe no Set. É muito rápido - O(1) de complexidade.

Exemplo 4: Verificando existência

const colors = new Set(["red", "green", "blue"]);

if (colors.has("red")) {
  console.log("Vermelho encontrado!");
}

console.log(colors.has("yellow")); // false

Método values()

O método values() retorna um novo Iterator contendo os valores do Set na ordem de inserção.

Exemplo 5: Iterando com values()

const letters = new Set(["a", "b", "c"]);
const iterator = letters.values();

for (const value of iterator) {
  console.log(value);
}

Método keys()

O método keys() é idêntico a values(). Existe apenas para compatibilidade com Map.

Exemplo 6: keys() é igual a values()

const mySet = new Set([1, 2, 3]);

// keys() retorna os mesmos valores que values()
for (const key of mySet.keys()) {
  console.log(key);
}

Método entries()

O método entries() retorna um Iterator com pares [value, value]. O valor é repetido para compatibilidade com Map.

Exemplo 7: Usando entries()

const mySet = new Set(["x", "y", "z"]);

for (const entry of mySet.entries()) {
  console.log(entry);
}
// ["x", "x"]
// ["y", "y"]
// ["z", "z"]

Método forEach()

O método forEach() executa uma função para cada elemento do Set.

Exemplo 8: Usando forEach()

const numbers = new Set([10, 20, 30]);
let sum = 0;

numbers.forEach(function(value) {
  sum += value;
});

console.log(sum); // 60

Referência de Métodos

Método Descrição Retorno
add(value) Adiciona um elemento ao Set O próprio Set
delete(value) Remove um elemento do Set true/false
has(value) Verifica se valor existe true/false
clear() Remove todos os elementos undefined
values() Retorna Iterator de valores Iterator
keys() Igual a values() Iterator
entries() Retorna Iterator [value, value] Iterator
forEach(callback) Executa função para cada elemento undefined

Propriedade size

A propriedade size retorna o número de elementos no Set.

Nota: Diferente de Arrays que usam length, Sets usam size. 

const mySet = new Set([1, 2, 3]);
console.log(mySet.size); // 3
