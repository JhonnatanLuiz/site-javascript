JavaScript Maps

Um Map é uma coleção de pares chave → valor. Diferente de objetos comuns, as chaves de um Map podem ser qualquer tipo (objetos, funções, números, etc.).

Quando usar Map? Quando você precisa de chaves não-string, ordem de inserção previsível e operações claras para coleção chave-valor.

Map vs Object

Característica Map Object
Tipo de chave Qualquer tipo String/Symbol (na prática)
Tamanho map.size Precisa calcular
Ordem Ordem de inserção Depende / regras próprias
Iteração Simples (entries/keys/values) for...in / Object.keys()

Exemplos Interativos

Clique em cada demo para ver o resultado.

Demo 1: Criando um Map e usando set/get

const usuarios = new Map();
usuarios.set('admin', 'Jhonnatan');
usuarios.set('editor', 'Ana');

usuarios.get('admin'); // "Jhonnatan"
usuarios.size;        // 2

Demo 2: Chaves de qualquer tipo

const m = new Map();
const objKey = { id: 1 };
m.set(objKey, 'valor para objeto');
m.set(10, 'dez');
m.set(true, 'sim');

Demo 3: Iterando com entries()

const precos = new Map([
  ['café', 8],
  ['pão', 2],
  ['leite', 6]
]);

for (const [produto, preco] of precos.entries()) {
  // ...
}

Demo 4: Convertendo Map ↔ Array

const m = new Map([['a', 1], ['b', 2]]);
const arr = Array.from(m); // [['a',1], ['b',2]]
const m2 = new Map(arr);   // volta a ser Map
