JavaScript Syntax

Regras de Sintaxe

A sintaxe são as regras de como os programas devem ser construídos:

Exemplo de Sintaxe

// Como declarar variáveis:
let x = 5;
let y = 6;

// Como computar valores:
let z = x + y;

// Eu sou um Comentário. Não faço nada.

Valores JavaScript

A sintaxe JavaScript define dois tipos de valores:

📌

Literais (Valores Fixos)

Valores que não mudam, como números e strings

📦

Variáveis (Valores Variáveis)

Contêineres que armazenam dados que podem mudar

Literais JavaScript

As regras de sintaxe mais importantes para literais (valores fixos) são:

Números

Números são escritos com ou sem decimais:

10.50
1001

Resultado: 10.50 e 1001

Strings (Textos)

Strings são textos, escritos entre aspas duplas ou simples:

"John Doe"
'John Doe'

Resultado: John Doe

Palavras-Chave JavaScript

Palavras-chave JavaScript são usadas para definir ações a serem executadas.

As palavras-chave let e const criam variáveis:

Exemplo - Keywords

let x = 5;
const fname = "John";

⚠️ Nota: Palavras-chave JavaScript são case-sensitive (sensíveis a maiúsculas/minúsculas).

JavaScript não interpreta LET ou Let como a palavra-chave let.

Variáveis JavaScript

Variáveis são contêineres para armazenar valores de dados.

Variáveis devem ser identificadas com nomes únicos.

Exemplo - Variáveis

// Define x como uma variável
let x;

// Atribui o valor 6 para x
x = 6;

Identificadores JavaScript

Um identificador é o nome que você dá a uma variável.

Regras para identificadores:

  • ✓ Devem começar com uma letra, _, ou $
  • ✓ Podem conter dígitos após o primeiro caractere
  • ✗ Não podem ser uma palavra-chave reservada (let, const, if, etc.)
  • ⚠ São case-sensitive (maiúsculas e minúsculas são diferentes)

✓ Nomes Válidos

  • • myVariable
  • • _private
  • • $price
  • • user123
  • • firstName

✗ Nomes Inválidos

  • • 123user (começa com número)
  • • my-variable (hífen não permitido)
  • • let (palavra reservada)
  • • first name (espaço não permitido)
  • • @price (@ não permitido)

Operadores JavaScript

JavaScript usa o operador de atribuição = para atribuir valores a variáveis:

Operador de Atribuição

let x = 5;
let y = 6;
let sum = x + y;

JavaScript usa operadores aritméticos (+ - * /) para computar valores:

Operadores Aritméticos

5 * 10

Expressões JavaScript

Uma expressão é uma combinação de valores, variáveis e operadores, que computa um valor.

Exemplo 1: Expressão Numérica

(5 + 6) * 10 resulta em 110:

(5 + 6) * 10

Exemplo 2: Expressão com Variáveis

Expressões também podem conter variáveis:

let x = 10;
x * 10;

Exemplo 3: Expressão com Strings

"John" + " " + "Doe" resulta em "John Doe":

"John" + " " + "Doe"

JavaScript é Case Sensitive

Identificadores JavaScript são sensíveis a maiúsculas/minúsculas.

As variáveis lastName e lastname são variáveis diferentes:

Exemplo - Case Sensitive

let lastName = "Doe";
let lastname = "Peterson";

JavaScript e Camel Case

Historicamente, programadores usaram diferentes formas de juntar múltiplas palavras em um nome de variável:

Estilo Exemplo Status em JS
Hífens first-name, last-name ❌ NÃO permitido
Underscore first_name, last_name ⚠️ Permitido
Upper Camel Case
(Pascal Case)		 FirstName, LastName ✓ Permitido
Lower Camel Case firstName, lastName ✅ RECOMENDADO

💡 Convenção JavaScript:

Programadores JavaScript tendem a usar lower camel case (camelCase). Comece com letra minúscula e capitalize as primeiras letras das palavras subsequentes.

Exemplo: myVariableName, calculateTotalPrice
