JavaScript Iterables

Um iterável é um valor que pode ser percorrido em sequência. Na prática, é algo que funciona com for...of e com o operador spread.

Como reconhecer: iteráveis expõem Symbol.iterator (um método que devolve um iterador).

Iteráveis comuns

✅ Iteráveis por padrão

  • Arrays
  • Strings
  • Map / Set
  • NodeList (em muitos casos)

⚠️ Não iteráveis (geralmente)

  • Objetos literais (ex.: {a:1})
  • Numbers e booleans
  • null / undefined
