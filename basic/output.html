JavaScript Output

Possibilidades de Exibição JavaScript

JavaScript pode "exibir" dados de diferentes maneiras:

📝 innerHTML

Escrever em um elemento HTML

📄 document.write()

Escrever na saída HTML

⚠️ window.alert()

Escrever em uma caixa de alerta

🔍 console.log()

Escrever no console do navegador

Usando innerHTML

Para acessar um elemento HTML, você pode usar o método document.getElementById(id).

Use o atributo id para identificar o elemento HTML.

Em seguida, use a propriedade innerHTML para alterar o conteúdo HTML do elemento:

Exemplo - innerHTML

<!DOCTYPE html>
<html>
<body>

<h1>Minha Primeira Página Web</h1>
<p>Meu Primeiro Parágrafo</p>
<p id="demo"></p>

<script>
document.getElementById("demo").innerHTML = "<h2>Hello World</h2>";
</script>

</body>
</html>
Clique no botão acima!

💡 Nota: Alterar a propriedade innerHTML de um elemento HTML é a maneira mais comum de exibir dados em HTML.

Usando innerText

Use a propriedade innerText para alterar o texto interno de um elemento HTML:

Exemplo - innerText

document.getElementById("demo").innerText = "Hello World";
Clique no botão acima!

📌 Diferença:

  • • Use innerHTML quando você quiser alterar um elemento HTML (pode conter tags HTML).
  • • Use innerText quando você quiser alterar apenas o texto simples (sem HTML).

Usando document.write()

Para fins de teste, é conveniente usar document.write():

Exemplo - document.write()

<!DOCTYPE html>
<html>
<body>

<h1>Minha Primeira Página Web</h1>
<p>Meu primeiro parágrafo.</p>

<script>
document.write(5 + 6);
</script>

</body>
</html>

Resultado simulado:

Minha Primeira Página Web

Meu primeiro parágrafo.

11

⚠️ AVISO IMPORTANTE:

Usar document.write() após um documento HTML ser carregado, irá deletar todo o HTML existente:

O método document.write() deve ser usado apenas para testes.

Usando window.alert()

Você pode usar uma caixa de alerta para exibir dados:

Exemplo - window.alert()

window.alert(5 + 6);

💡 Dica: Você pode pular a palavra-chave window.

Em JavaScript, o objeto window é o escopo global. Isso significa que variáveis, propriedades e métodos por padrão pertencem ao objeto window. Isso também significa que especificar a palavra-chave window é opcional:

alert(5 + 6);  // Também funciona!

Usando console.log()

Para fins de depuração, você pode chamar o método console.log() no navegador para exibir dados.

Você aprenderá mais sobre depuração em um capítulo posterior.

Exemplo - console.log()

console.log(5 + 6);

// Console do Navegador (F12)

> 11

🔍 Dica de Desenvolvedor: Pressione F12 no seu navegador para abrir as Ferramentas de Desenvolvedor e ver a aba Console.

JavaScript Print

JavaScript não tem nenhum objeto de impressão ou métodos de impressão.

Você não pode acessar dispositivos de saída a partir do JavaScript.

A única exceção é que você pode chamar o método window.print() no navegador para imprimir o conteúdo da janela atual.

Exemplo - window.print()

<button onclick="window.print()">Imprimir esta página</button>

📋 Resumo dos Métodos de Saída

Método Descrição Uso Comum
innerHTML Altera conteúdo HTML de elemento ✅ Produção (mais usado)
innerText Altera texto simples de elemento ✅ Produção
document.write() Escreve diretamente no documento ⚠️ Apenas testes
window.alert() Exibe caixa de alerta ⚠️ Mensagens/Avisos
console.log() Escreve no console do navegador 🔍 Depuração/Debug
window.print() Imprime a página 🖨️ Impressão
