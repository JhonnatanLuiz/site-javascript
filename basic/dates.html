JavaScript Dates

Objetos Date em JavaScript permitem trabalhar com datas e horários.

Exemplo - Data Atual

const d = new Date();
document.getElementById("demo1").innerHTML = d;

Criando Objetos Date

Objetos Date são criados com o construtor new Date().

Existem 9 formas de criar um novo objeto Date:

new Date()
new Date(date string)

new Date(year, month)
new Date(year, month, day)
new Date(year, month, day, hours)
new Date(year, month, day, hours, minutes)
new Date(year, month, day, hours, minutes, seconds)
new Date(year, month, day, hours, minutes, seconds, ms)

new Date(milliseconds)

💡 Nota: Os parâmetros de data são baseados em zero. Janeiro é 0, Dezembro é 11.

new Date()

new Date() cria um objeto com a data e hora atuais:

Exemplo

const d = new Date();

new Date(date string)

new Date(date string) cria um objeto a partir de uma string de data:

Exemplo

const d = new Date("October 13, 2014 11:13:00");

new Date(year, month, ...)

new Date(year, month, ...) cria um objeto com uma data especificada:

Exemplo - 7 parâmetros

const d = new Date(2018, 11, 24, 10, 33, 30, 0);

Exemplo - 6 parâmetros (sem ms)

const d = new Date(2018, 11, 24, 10, 33, 30);

⚠️ JavaScript conta meses de 0 a 11:

Janeiro = 0, Dezembro = 11.

Século Anterior

Anos com um ou dois dígitos serão interpretados como 19xx:

Exemplo

const d = new Date(99, 11, 24);

new Date(milliseconds)

new Date(milliseconds) cria um objeto a partir do zero time mais milliseconds:

Exemplo

const d = new Date(100000000000);

💡 Zero time é 01 Janeiro 1970 00:00:00 UTC.

100.000.000.000 milliseconds é aproximadamente 03 Março 1973.

Exibindo Datas

Por padrão, JavaScript exibirá datas no formato de string completo:

Mon Nov 25 2025 15:30:00 GMT-0300 (Brasília Standard Time)

Quando você exibe um objeto Date em HTML, ele é automaticamente convertido para string com o método toString().
