JavaScript Random

Em JavaScript, a forma padrão de gerar números pseudo-aleatórios é Math.random(). Ela retorna um número no intervalo [0, 1) (inclui 0, mas nunca chega em 1).

W3Schools: JavaScript Random

Fórmulas essenciais

  • Math.random() → [0, 1)
  • Math.floor(Math.random() * 10) → inteiro de 0 a 9
  • Math.floor(Math.random() * (max - min + 1)) + min → inteiro min..max (inclusive)
  • array[Math.floor(Math.random() * array.length)] → item aleatório do array
